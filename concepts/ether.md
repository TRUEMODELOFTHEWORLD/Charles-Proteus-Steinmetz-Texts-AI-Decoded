# Ether

Status: seeded; high caution.

## Core Historical Meaning

In `Radiation, Light and Illumination`, Steinmetz uses ether as the hypothesized medium of light wave motion. This is historically real language in the source.

## Modern Equivalent

There is no one-to-one modern equivalent. Standard physics uses electromagnetic fields and spacetime, not a mechanical luminiferous ether.

## Interpretive Boundary

It is valid to document Steinmetz's ether statements. It is also valid to create labeled interpretive readings. It is not valid to claim Steinmetz proved a later ether theory unless the source evidence supports that claim.

## Research Questions

- Does Steinmetz use ether language consistently across books?
- Does he treat ether as mechanical substance, mathematical field medium, or inherited vocabulary?
- Did later editions change this language?
