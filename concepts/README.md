# Concept Encyclopedia

This directory holds cross-source concept pages. Source-specific concept work begins in `analysis/<source-id>/concepts/`; mature concepts should be promoted here after cross-source review.

Each page should show:

- Steinmetz's own definition or usage.
- Source appearances and citations.
- Modern interpretation.
- Mathematical explanation.
- Diagrammatic explanation.
- Tesla-era comparison when relevant.
- Clearly labeled ether-field interpretive reading.
- Current status in modern engineering language.
