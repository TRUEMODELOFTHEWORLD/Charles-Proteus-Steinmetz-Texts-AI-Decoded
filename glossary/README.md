# Glossary

The glossary translates historical electrical language into modern terms without erasing the original conceptual world.

Each term should include:

- Original term.
- Steinmetz usage.
- Modern equivalent.
- Mathematical meaning.
- Conceptual meaning.
- Current status.
- Example quote.
- Related concepts.
