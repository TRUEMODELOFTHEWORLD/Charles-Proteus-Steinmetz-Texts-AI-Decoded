# Ultra-Red

Status: seeded.

## Original Term

Ultra-red.

## Modern Equivalent

Infrared.

## Steinmetz Usage

Steinmetz uses ultra-red for radiation below the visible red region, on the lower-frequency, longer-wavelength side of visible light.

## Status

Renamed in standard modern usage.

## Related Concepts

- Radiation spectrum
- Temperature radiation
- Visible light
