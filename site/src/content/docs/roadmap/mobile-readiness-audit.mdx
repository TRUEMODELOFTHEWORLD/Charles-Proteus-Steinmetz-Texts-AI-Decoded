## Manual Review Rules

- No page-level horizontal scroll.
- Generated source text expands naturally or wraps without clipped lines.
- Tables scroll inside their own border when too wide.
- Bottom reader controls do not hide navigation, citations, or next/previous links.
- Source-only filtering, page search, translation shortcuts, and lightbox controls remain reachable by touch.
- Diagram, scan, and tool panels fit without overlapping adjacent text.

## Publication Rule

Any newly generated page family that introduces wide tables, source transcripts, scan embeds, visual galleries, or tool canvases should either use the existing mobile-safe classes or extend this audit before publication.
