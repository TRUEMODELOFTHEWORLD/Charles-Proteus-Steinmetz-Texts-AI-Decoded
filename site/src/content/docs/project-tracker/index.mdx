This page mirrors the repo-level `MASTER_PROJECT_TRACKER.md`. It exists so readers, contributors, and reviewers can see the whole build path: what is already public, what is candidate, what needs verification, and what comes next.

The rule is simple: move quickly, but label the layer. A page may be useful before it is final, but source fact, modern translation, mathematical reconstruction, historical note, interpretive reading, speculative connection, and needs-verification material must remain distinct.

## Future Scope

The archive may later expand into a wider electrical-science codex including Nikola Tesla, Eric Dollard, Walter Russell, and other researchers. That wider design should keep Steinmetz as a distinct source domain and use comparison pages instead of blending different authors into one voice.

The architecture note is public at [Future Codex Architecture](/Charles-Proteus-Steinmetz-Texts-AI-Decoded/roadmap/future-codex-architecture/). It is recorded now so future expansion does not blur Steinmetz-first source work.
