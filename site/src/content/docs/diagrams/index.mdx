## Status

The original crops above are scan-derived assets with crop manifests and checksums. The recreated graphics below are still reading aids rather than documentary evidence. The long-term goal is to place original scan crops, cleaned versions, vector recreations, annotations, equations, and concept links side by side.

## Planned Workflow

1. Extract original page images.
2. Crop figures.
3. Save original crops.
4. Clean contrast and labels.
5. Recreate vector forms when useful.
6. Add modern annotations.
7. Add separate ether-field annotations only when useful.
