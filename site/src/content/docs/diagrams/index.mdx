## Status

The diagrams above are recreated guides for navigation and explanation. They are not yet original scan crops. The next image stage is to extract original Steinmetz figures from source scans, clean them, and place the original beside the recreated version. Until that step is complete, every recreated graphic should be treated as a reading aid rather than documentary evidence.

2. Crop figures.
3. Save original crops.
4. Clean contrast and labels.
5. Recreate vector forms when useful.
6. Add modern annotations.
7. Add separate ether-field annotations only when useful.
