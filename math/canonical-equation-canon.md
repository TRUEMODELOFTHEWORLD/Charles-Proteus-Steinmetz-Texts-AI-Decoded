# First Canonical Equation Set

This file names the first twelve equations that should become the public mathematical spine of the Steinmetz archive.

Status rule: these are source-located candidates unless otherwise stated. They are important enough to organize and explain now, but exact typography, page numbers, and OCR defects still need scan verification before final canonical status.

## Canon Set

1. `S = f lambda` - velocity, frequency, and wave length.
2. `I = i + ji'` - rectangular complex quantity.
3. `j^2 = -1` - quarter-period operator.
4. `x = 2 pi f L` - inductive reactance.
5. `x_1 = 1 / (2 pi f C)` - condensive reactance.
6. `Z = r + jx` - complex impedance.
7. `E = ZI` - symbolic Ohm's law.
8. `Y = 1 / Z = g - jb` - admittance.
9. `g = r / (r^2 + x^2)` and `b = x / (r^2 + x^2)` - admittance components.
10. `P = EI cos theta` - AC power equation.
11. `P = I^2 r_eff` - effective resistance as real-power coefficient.
12. `b = 2 pi f C` - capacity susceptance.
