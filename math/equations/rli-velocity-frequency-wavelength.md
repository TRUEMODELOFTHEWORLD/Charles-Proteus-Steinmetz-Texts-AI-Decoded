## Original Context

Steinmetz uses the relation between propagation speed, frequency, and wavelength to compare visible light, electric waves, and alternating-current fields.

## Modern Notation

```math
S = f \lambda
```

```math
f = \frac{S}{\lambda}
```

```math
\lambda = \frac{S}{f}
```

## Variables

- `S`: propagation speed of radiation.
- `f`: frequency in cycles per second, now hertz.
- `lambda`: wavelength.

## Worked Examples

Visible radiation example:

```math
S = 3 \times 10^{10}\ \text{cm/s}
```

```math
\lambda \approx 60 \times 10^{-8}\ \text{cm}
```

```math
f \approx 5 \times 10^{14}\ \text{s}^{-1}
```

Sixty-cycle AC field example:

```math
\lambda = \frac{3 \times 10^{10}}{60}
  = 5 \times 10^8\ \text{cm}
```

This is about 5,000 km or 3,100 miles.

## Physical Meaning

The formula makes Steinmetz's unifying scale possible: ordinary AC, wireless waves, Hertzian waves, infrared, visible light, ultraviolet, and X-rays are all arranged by frequency and wavelength.

## Modern Status

Still fundamental. Modern notation uses `c` for the speed of light in vacuum and `f` in hertz.
