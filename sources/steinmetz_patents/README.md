# Steinmetz Patent Register

This folder is the custody and control layer for Charles Proteus Steinmetz patent work.

The current register is intentionally marked `seeded_incomplete`. Wikipedia states that Steinmetz held over 200 patents, while the current Wikipedia bibliography lists only selected examples. The archive must not call this complete until an authority pass against USPTO, PatentCenter, Google Patents, Espacenet, and assignment data has been completed.

## Processing Rule

Each mature patent record should eventually include:

- patent number, title, inventor, assignee, filing date, publication date, and authority links
- original patent PDF and checksum
- extracted patent drawings and drawing captions
- OCR text and corrected claim text
- apparatus summary in modern engineering language
- mathematical content, if present
- relation to Steinmetz books, papers, concepts, equations, and diagrams
- comparison notes to modern terminology
- interpretive notes only when visibly labeled as interpretation

Patent claims are legal-apparatus claims. They should never be used as proof of Steinmetz's complete scientific ontology unless supported by his technical writings.
